Troubleshooting
1
Host via Local Server

To use auto-refresh, start a simple HTTP server in the project folder: 

python3 -m http.server 8000

Then open: http://localhost:8000/mic/train_dashboard/

2
Direct CSV Reading

If you don't use a web server, you can drag and drop your file into this screen anytime to update the metrics.
