Test Protected Endpoints

Select an API rule configuration to request from port 5006
GET /api/test-auth/public Public (Allows anonymous access)
GET /api/test-auth/secure Requires [AppAuthorize] (Any valid token)
GET /api/test-auth/admin-role Requires [AppAuthorize(Roles = "Admin")]
GET /api/test-auth/admin-claim Requires claim: be.admin ("admin" param)
GET /api/test-auth/custom-claim Requires claim: be.test.read ("test.read" param)
GET /api/test-auth/acl-read/{id} Requires Action.Read on ResourceType
GET /api/test-auth/acl-write/{id} Requires Action.Write on ResourceType
GET /api/test-auth/user-info Inspect current token claims list
