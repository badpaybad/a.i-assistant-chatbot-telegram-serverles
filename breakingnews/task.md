1. Vào sources.md để đọc tin hàng ngày
2. Tổng hợp nhanh các tin nổi bật lưu mỗi source thành file .txt định dạng tên file {tên source}_{năm_tháng_ngày}.txt
3. Lưu các tổng hợp nhanh vào file .txt định dạng tên file news_{năm_tháng_ngày}.txt
4. Đưa ra nhận định xu hướng các thông tin vào file .txt định dạng tên file trend_{năm_tháng_ngày}.txt
5. Phân loại nội dung và lấy thông tin theo mô tả ở ner.md, tạo file .csv định dạng tên file data_{năm_tháng_ngày}.csv
