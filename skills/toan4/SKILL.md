# Hướng dẫn sử dụng Skill toan4

Khi người dùng cung cấp một bức ảnh bài tập toán lớp 4 tiếng Việt và yêu cầu dùng `skill toan4`, bạn hãy tuân thủ các tài liệu có trong thư mục này:

1. Đọc file `plan.md` để hiểu ý tưởng cốt lõi, từ việc phân tích hình ảnh, thiết kế giao diện game tương tác, đến cách thiết lập cơ chế phản hồi đúng/sai.
2. Đọc và thực thi từng bước trong file `task.md` để triển khai thành một folder game thực tế tương ứng với tên bức ảnh truyền vào. Game phải được thiết kế bằng HTML5/CSS/JS sinh động, bắt mắt và phù hợp với học sinh lớp 4.
3. Tạo thư mục kết quả (ví dụ: `games/toanlop4/<tên_ảnh>`) với các thành phần `index.html`, `style.css` và `script.js` theo đúng yêu cầu đề ra.
4. Trả về cho người dùng đường dẫn hoặc cách mở file để trải nghiệm.
