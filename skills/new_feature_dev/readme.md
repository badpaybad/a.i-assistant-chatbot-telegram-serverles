# Skill: Phát triển nghiệp vụ mới (New Feature Development)

Dùng để tự động hóa quy trình phát triển một nghiệp vụ hoặc tính năng mới trong dự án TreeOfThought.

## Các chức năng chính

1. Tạo subfolder nghiệp vụ trong `TreeOfThought/docs`.
2. Tạo file `whattodo.md` để tiếp nhận yêu cầu từ người dùng.
3. Tự động đọc `whattodo.md` và viết giải pháp vào `howtodo.md`.
4. Khởi tạo project Backend (C#) trong `TreeOfThought/backend`.
5. Khởi tạo folder Frontend trong `TreeOfThought/frontend/web/src/modules`.

## Cách sử dụng

Người dùng có thể yêu cầu: "Tạo nghiệp vụ mới tên là Quản lý kho" hoặc "Phát triển tính năng chat realtime".
Skill này sẽ tự động thực hiện các bước setup ban đầu.
