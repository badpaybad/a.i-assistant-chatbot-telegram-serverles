# Note for GPU Acceleration support:
#
# 1. NVIDIA CUDA GPU:
#    The standard torch/torchvision packages usually auto-detect NVIDIA CUDA.
#
# 2. AMD GPU (ROCm) on Linux:
#    To train using AMD GPUs (e.g. Radeon 780M / consumer RDNA3 / RX 6000 & 7000 series):
#    - First, uninstall standard packages:
#      pip uninstall -y torch torchvision torchaudio
#    - Install the ROCm enabled packages (e.g. ROCm 6.0):
#      pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/rocm6.0

opencv-python
